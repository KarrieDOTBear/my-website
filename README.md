<h1>Project: DIY Website Using GitHub Pages</h1>

<p>My name is Karrie, and this repository holds my first attempt at publishing a website without using a platform that does everything for me. The project includes two HTML pages. The first is my original photo essay, and the second is a reflection on what it was like to get this whole thing online.</p>

<p>I created this repository to host my files through GitHub Pages. The main page is named index.html, since GitHub will not load the site correctly without that. My images are stored in the same repository, so the links stay simple.</p>

<p>This project was my introduction to GitHub. I am still getting used to how the site organizes files and how small mistakes can break things in ways that are not obvious at first. Even so, it was interesting to see how much control you have when you are not relying on a publishing platform.</p>

<p>The second HTML page contains my reflection on the process. I linked it from the main page so the site feels connected.</p>

<p>That is the basic layout of this repository. It is simple, but it works, and getting it to work taught me more than I expected.</p>
